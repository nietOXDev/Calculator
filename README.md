# Calculator

## Overview
This is a simple web-based calculator built to perform basic arithmetic operations. The project demonstrates the use of HTML, CSS, and JavaScript to create an interactive user interface and handle user input dynamically.

## Features
- Addition, subtraction, multiplication, and division
- Responsive design for different screen sizes
- Clear and easy-to-use interface
- Button-based input for user-friendly interaction

## Technologies Used
- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for logic and dynamic interaction

## How to Use
1. Clone or download the repository.
2. Open the `index.html` file in a web browser.
3. Click on the buttons to input numbers and operations.
4. Press `=` to calculate the result or `C` to clear the input.

## Project Purpose
This project was created to practice front-end web development skills, including DOM manipulation, event handling, and responsive design.
